# steady-fetch Decentralization Manifesto

steady-fetch is not a product. It is a decentralized protocol for HTTP resilience. No single entity controls it. No company can shut it down.

## Why This Is Unstoppable
- No central server
- The network belongs to everyone
- The only way to stop steady-fetch is to shut down the entire internet

## How To Join
1. Install steady-fetch
2. Enable P2P: steady({ p2p: { enable: true } })
3. Your instance will discover peers and join the mesh automatically
